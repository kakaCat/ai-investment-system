# Sprint 状态分析

分析当前 Sprint 的进度、风险和阻塞项。

---

请分析当前 Sprint 状态：

## 分析内容

### 1. 任务进度分析
从 `management/sprints/current.md` 读取：
- 总任务数和完成情况
- 各角色任务分布（BE/FE/QA）
- 延期任务识别

### 2. 依赖关系分析
- 任务之间的依赖关系
- 阻塞链识别
- 关键路径分析

### 3. 风险评估
- 延期风险（基于进度）
- 技术风险（基于复杂度）
- 依赖风险（外部依赖）

### 4. 工作量分析
- 剩余工作量估算
- 资源分配建议
- 是否需要调整范围
