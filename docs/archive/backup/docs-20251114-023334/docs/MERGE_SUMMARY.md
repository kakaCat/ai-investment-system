## ✨ v3.1 新增功能

### 2.9 事件分析与追踪

#### 1. 事件类型体系
**四大类事件、16种子类型**:
- **政策事件** (4个子类型)
  - 货币政策、财政政策、监管政策、国际政策
- **公司事件** (4个子类型)
  - 财报相关、公司治理、资本运作、业务变化
- **市场事件** (4个子类型)
  - 指数调整、资金流动、市场情绪、技术突破
- **行业事件** (4个子类型)
  - 技术突破、竞争格局、供需关系、行业政策

#### 2. 核心功能
- ✅ **事件记录功能**
  - 手动记录事件(详细表单)
  - 批量导入(CSV/MCP自动采集)
  - 幂等性保证(防重复)

- ✅ **事件时间线**
  - 全局事件时间线(筛选、分类)
  - 股票事件时间线(详情页展示)
  - 可视化展示(图标、颜色、重要性)

- ✅ **AI事件影响分析**
  - 单个事件影响评估(全球市场/行业/个股)
  - 批量事件综合影响分析
  - 预测价格变化(带置信度)

- ✅ **AI分析集成**
  - 单股分析增强(引用近30天事件)
  - 持仓分析增强(事件告警、影响矩阵)
  - 基于事件的操作建议

- ✅ **事件提醒与日历**
  - 重要事件日历(宏观数据发布日)
  - 持仓相关事件日历(财报、分红)
  - 自动监控与推送(Critical/High事件)

- ✅ **数据管理**
  - 多数据源(MCP自动采集 + 用户手动添加)
  - 数据质量保证(去重、验证、AI评估)
  - 存储策略(热/温/冷数据分层)

- ✅ **定时任务**
  - 事件收集任务(每小时)
  - 事件影响分析任务(异步)
  - 事件提醒任务(每日)

- ✅ **前端UI设计**
  - 事件中心页面
  - 股票详情页事件区块
  - 持仓分析页事件告警

- ✅ **技术优化**
  - 性能优化(分区表、GIN索引、Redis缓存)
  - AI成本控制(批量分析、缓存)
  - 数据质量回测

## 🎯 设计亮点

### 1. 完整的事件体系
✅ **四层事件分类**: 政策/公司/市场/行业 → 16个子类型
✅ **五级重要性**: Critical → High → Medium → Low → Very Low
✅ **影响量化**: 方向(利好/利空/中性/复杂) + 强度(0-100)

### 2. AI驱动的智能分析
✅ **自动影响评估**: AI评估事件对市场/行业/个股的影响
✅ **预测价格变化**: 预期变化百分比 + 置信度
✅ **综合影响分析**: 多事件叠加影响评估

### 3. 全场景深度集成
✅ **单股分析**: 展示近30天相关事件 + 事件驱动建议
✅ **持仓分析**: 事件告警 + 影响矩阵 + 组合层面建议
✅ **策略生成**: 基于事件的操作计划

### 4. 用户体验优化
✅ **时间线可视化**: 直观展示事件时间线
✅ **智能提醒**: 重要事件/财报日期提前提醒
✅ **事件日历**: 一目了然的重要事件日程

### 5. 技术架构完整
✅ **EventAdapter**: 从MCP获取事件
✅ **定时任务**: 自动采集、分析、提醒
✅ **性能优化**: 缓存、分区、批量处理
✅ **数据质量**: 去重、验证、回测
