## 功能模块说明

### 1. 筛选器

**筛选维度**:
- **时间范围**: 今天/近7天/近30天/近90天/自定义
- **事件类型**: 全部/政策/公司/市场/行业
- **重要性**: 全部/Critical/High/Medium/Low
- **影响范围**: 全部/全球/特定市场/特定行业/特定股票
- **市场**: 全部/A股/港股/美股
- **我的持仓**: 仅显示影响持仓的事件

**交互**:
- 多选/单选支持
- [重置筛选] 清空所有筛选条件
- 实时显示筛选结果数量

### 2. 事件统计概览

**统计卡片**:
- 总事件数
- 未读事件数（红点标识）
- 重要事件数（Critical + High）
- 今日新增事件数

**交互**:
- 点击卡片快速应用对应筛选

### 3. 事件时间线

**视图切换**:
- **列表视图**: 按时间倒序列表（默认）
- **时间轴视图**: 垂直时间轴展示
- **日历视图**: 日历格式，点击日期查看

**事件卡片结构**:
```
┌─────────────────────────────────────┐
│ 🔴 时间  重要性  |  事件分类         │
│                                      │
│ 事件标题                             │
│                                      │
│ 📊 影响评分                          │
│ 🌍/🎯 影响范围                       │
│ 📈 市场/股价反应                     │
│                                      │
│ 💼 对您持仓的影响（如有）            │
│                                      │
│ 💡 AI建议                            │
│                                      │
│ [操作按钮...]                        │
└─────────────────────────────────────┘
```

**卡片配色**:
- 🔴 红色边框: Critical
- 🟠 橙色边框: High
- 🟡 黄色边框: Medium
- 🟢 绿色边框: Low

**排序选项**:
- 时间倒序（最新在前）
- 时间正序（最早在前）
- 重要性排序
- 影响力排序

### 4. 事件详情展示

**必需信息**:
- 事件时间
- 重要性级别
- 事件分类（类型 > 子类型）
- 事件标题/摘要
- AI影响分析

**可选信息**（根据事件类型）:
- 影响的股票/行业/市场
- 市场/股价实际反应
- 对用户持仓的具体影响
- AI操作建议
- 新闻来源链接

**操作按钮**:
- [查看完整分析]: 展开详细的AI分析报告
- [标记已读]: 标记该事件
- [添加到策略]: 将事件关联到投资策略
- [分享]: 分享事件卡片
- [查看股票详情]: 跳转到股票详情页

### 5. 批量操作

**选择机制**:
- 单个事件卡片左上角复选框
- [全选当前页] 按钮
- 已选中数量实时显示

**批量操作**:
- [批量AI分析]: 对选中的多个事件进行综合影响分析
- [批量标记已读]: 一键标记
- [导出选中]: 导出为PDF/Excel
- [添加到日历]: 添加到事件日历

## 数据刷新策略

1. **进入页面时**: 加载最新事件
2. **定时刷新** (每10分钟): 检查新事件
3. **手动刷新**: 下拉刷新
4. **实时推送** (WebSocket): Critical事件立即推送

---

## 状态处理

### 加载状态
- 首次加载: 骨架屏
- 加载更多: 加载动画

### 空状态
- 无事件: "暂无事件，系统将自动采集最新事件"
- 筛选无结果: "未找到符合条件的事件，尝试调整筛选条件"

### 错误状态
- 加载失败: 重试按钮
- AI分析失败: 错误提示
