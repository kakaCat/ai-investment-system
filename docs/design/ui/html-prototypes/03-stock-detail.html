公司简介

青岛啤酒股份有限公司成立于1903年，是中国最早的啤酒生产企业之一。 公司主要从事啤酒的生产与销售，拥有青岛啤酒等知名品牌。 产品销往全球100多个国家和地区，是中国啤酒行业的龙头企业。
