# PRD v3.1 — AI驱动的个人投资管理系统

**版本**: v3.1
**日期**: 2025-01-15
**状态**: 设计完成，待实施

---

## 概述

基于 **事件驱动 + AI 赋能** 的个人投资管理系统，支持多账户、多市场（A股/港股/美股）的投资组合管理与策略复盘。

### 核心特点

- ✅ **多账户管理**：统一管理多个券商账户
- ✅ **精准盈亏**：分账户、分股票的精确盈亏计算
- ✅ **AI 策略生成**：基于 DeepSeek API 的投资建议与复盘分析
- ✅ **事件分析系统**（v3.1 核心）：4大类16子类事件，AI影响评估，持仓关联
- ✅ **策略回顾**：记录与追踪 AI 建议的执行情况与准确率
- ✅ **数据集成**：Tushare/AkShare 获取股票行情数据
